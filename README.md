Face Recognition Attendance System :


1. Overview : 

This project is a Face Recognition Based Attendance System built using Machine Learning (Artificial Neural Network) and Computer Vision.
It collects face images using a webcam, trains an ANN model, and recognizes individuals automatically.

The system demonstrates a complete ML pipeline:
Data Collection → Training → Face Recognition


2. Features

Real-time face detection using OpenCV

Dataset collection using webcam

Face recognition using ANN (MLPClassifier)

Supports multiple users

Automatic model training

Model saving using Pickle


3. Tech Stack

Language: Python

Libraries: OpenCV, NumPy, scikit-learn, Pickle

Concepts: Machine Learning, Artificial Neural Networks, Computer Vision


4. Output : 

Real-time face detection window

Image dataset generation

Model accuracy after training

Saved trained model (ann_model.pkl)
